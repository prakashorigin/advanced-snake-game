Q: What is this project about?
A: This is an Advanced Snake Game built using HTML5, CSS3, and JavaScript. It
   demonstrates professional web development including DOM manipulation, event
   handling, game mechanics, responsive design, and browser APIs.

Q: How does the game work?
A: The game runs a game loop every 100ms that:
   1. Moves the snake in the current direction
   2. Checks for food collision (eating)
   3. Checks for wall/self collision (game over)
   4. Updates the display
   User input (keyboard/touch) changes direction. Game becomes faster as score
   increases.

Q: What makes it "advanced"?
A: Advanced features include:
   - Progressive difficulty system (speed increases)
   - Level calculation (automatic every 5 points)
   - Mobile touch and swipe support
   - Pause/Resume functionality
   - High score persistence using localStorage
   - Professional UI with gradients and animations
   - Complete collision detection
   - Responsive design for all devices

Q: How did you run it?
A: I used Python's built-in HTTP server:
   python3 -m http.server 8000
   Then accessed it at http://localhost:8000
   This avoids CORS issues and simulates a real web server.

Q: Why not just open in browser?
A: Due to browser security (CORS policy), we need a server to run JavaScript
   properly. The HTTP server simulates production environment locally.

Q: How many lines of code?
A: Total: 1,500+ lines
   - HTML: 180+ lines
   - CSS: 500+ lines
   - JavaScript: 400+ lines
   - Documentation: 400+ lines

Q: What technologies are used?
A: HTML5 (structure), CSS3 (styling), JavaScript ES6+ (logic),
   LocalStorage API (data persistence), Touch Events (mobile),
   Python HTTP Server (local deployment).

PROBLEM: Game not loading at localhost:8000
SOLUTION:
1. Check server is running: look for "Serving HTTP..." in terminal
2. If not running, execute: python3 -m http.server 8000
3. Refresh browser: Ctrl+R or Cmd+R
4. Clear cache: Ctrl+Shift+R or Cmd+Shift+R
5. Try incognito/private mode
6. Check console for errors: F12 â†’ Console

PROBLEM: Touch buttons not responding on mobile
SOLUTION:
1. Ensure you're testing on actual phone (not desktop emulator)
2. Check JavaScript is enabled in browser settings
3. Try reloading page on mobile
4. Test on different mobile browser
5. Check console: F12 â†’ Console (on mobile too)

PROBLEM: High score not saving between sessions
SOLUTION:
1. Check localStorage is enabled: F12 â†’ Application â†’ Storage
2. Verify key exists: localStorage.getItem("snakeHighScore")
3. Check browser isn't in private/incognito mode
4. Ensure cookies are enabled in settings
5. Try clearing and restarting browser

PROBLEM: Server won't start (Address already in use)
SOLUTION:
1. Kill existing process: lsof -ti:8000 | xargs kill -9
2. Or use different port: python3 -m http.server 9000
3. Wait 10 seconds and try again
4. Restart terminal

PROBLEM: Game freezes or lags
SOLUTION:
1. Close other browser tabs
2. Restart browser
3. Clear browser cache
4. Check CPU usage (Activity Monitor)
5. Try different browser
6. Reduce other running applications

PROBLEM: Can't access game from mobile
SOLUTION:
1. Verify both devices on same WiFi: Settings â†’ WiFi
2. Check firewall settings (might block local traffic)
3. Get correct IP: ifconfig | grep "inet " | grep -v "127.0.0.1"
4. Try pinging Mac from mobile: ping 192.168.1.10
5. Make sure server is running: python3 -m http.server 8000
